# เรียนรู้พาสนุก 🎈

เว็บแอปเพื่อการเรียนรู้สำหรับเด็ก แตะแล้วมีเสียงอ่าน + โหมดเกมทายเสียง
มี 3 หมวด: **ก-ฮ** (พยัญชนะไทย 44 ตัว) • **สัตว์** • **ยานพาหนะ**
รองรับมือถือทุกขนาด เป็นไฟล์เดียว ไม่ต้องติดตั้งอะไร

## ✨ ฟีเจอร์
- 🗂️ **เมนูแยกกลุ่มชัดเจน** — หน้าเมนูหลักแบ่งเป็น 📚 เรียนรู้ และ 🎮 เกม & กิจกรรม เลือกหมวดแล้วมีปุ่มย้อนกลับ
- 📚 **หมวดเรียนรู้** — ก-ฮ, สระไทย, A-Z (เสียงอังกฤษ), เลขไทย ๑-๑๐ และเลข 1-10 (แสดงด้วยรูปนิ้วมือ), คำศัพท์ (การเดินทาง/ร่างกาย/บนโต๊ะอาหาร), สัตว์, ยานพาหนะ
- 🎮 **หมวดกิจกรรม** — กิจวัตรประจำวัน (กินข้าว/อาบน้ำ/แปรงฟัน/แต่งตัว/เข้าห้องน้ำ ฯลฯ) เป็นตัวการ์ตูนโต้ตอบ
- 🍳 **เกมทำอาหาร** — เล่นอิสระ เลือกวัตถุดิบ (ไทย+สากล) → เลือกวิธีทำ (ทอด/ต้ม/ย่าง/หั่นสด/ปั่น) → ปรุงเป็นเมนู (ข้าวผัด/ต้มยำ/พิซซ่า/เบอร์เกอร์ ฯลฯ หรือจานสร้างสรรค์เอง) → เลือก **กิน** หรือ **ขายได้เหรียญ** (สะสมเหรียญบันทึกไว้)
- 🚓 **เกมรถกับการใช้งาน** — ฟังหน้าที่ (ดับไฟ/จับผู้ร้าย/พาคนป่วย…) แล้วเลือกรถให้ถูก
- 🖌️ **เกมแต่งรถ** — ทาสีตัวรถ หลังคา และล้อ (เปลี่ยนล้อพร้อมกันเป็นสีเดียวกัน) เลือกได้ 11 สี + สุ่มสี — สอนเรื่องสีไปในตัว
- 🌈 **หมวดสี** — เรียนรู้สีต่างๆ ในกลุ่มเรียนรู้
- 🎬 **แอนิเมชันโต้ตอบเต็มจอ** — แตะการ์ดในโหมดเรียนรู้ ตัวการ์ตูนเด้งออกมาเต็มจอ **ขยับ/เดินไปมา** (สัตว์กระโดด/รถวิ่ง/ตัวอักษรลอย), แตะที่ตัว = เล่นแอนิเมชันโต้ตอบ (กระโดด/หมุน/โยกตัว) + ดาว-หัวใจกระเด็น + เปล่งเสียง, แตะขอบ = ปิด
- 🔊 **เสียงดังเต็มที่ + เอฟเฟกต์เสียง** — เสียงพูด volume สูงสุด พร้อมเสียง "ป๊อป/ดิ๊ง/บอยง" ประกอบ
- 🎲 **ดีไซน์ 3 มิติ** — การ์ดทรงของเล่นมีขอบลึก เงานุ่ม กดเด้ง + เอฟเฟกต์คอนเฟตตี/ดาวตอนตอบถูก
- 📚 **3 หมวดการเรียนรู้** — พยัญชนะ ก-ฮ (44), สัตว์ (60+), ยานพาหนะ (35+) สลับได้ด้วยแท็บ
- 📖 **โหมดเรียนรู้** — แตะเพื่อฟังเสียง (เช่น "กอ ไก่", "ช้าง", "รถยนต์") พร้อมรูปประกอบ (emoji)
- 🎮 **โหมดทายเสียง** — ฟังเสียงแล้วเลือกคำตอบที่ถูก
- 🏆 **ระบบคะแนน** — มีโบนัสตอบต่อเนื่อง + บันทึกคะแนนสูงสุด (best score) แยกแต่ละหมวด จำค่าไว้ในเครื่อง
- 📱 รองรับทุกขนาดจอ (มือถือ / แท็บเล็ต / คอม)
- 🔊 ใช้เสียงสังเคราะห์ในเครื่อง (Web Speech API) — ไม่ต้องมีไฟล์เสียง เล่นได้แม้ออฟไลน์

> หมายเหตุ: คุณภาพเสียงขึ้นกับอุปกรณ์ iOS/Android/Chrome มักมีเสียงภาษาไทยติดมาแล้ว
> ถ้าเครื่องไม่มีเสียงไทยติดตั้ง อาจไม่ได้ยินเสียง — ลองเพิ่มภาษาไทยในการตั้งค่าเครื่อง

## ▶️ ทดลองบนเครื่อง
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย (ดับเบิลคลิก)

## 🚀 นำขึ้น GitHub Pages
1. สร้าง repo ใหม่บน GitHub (เช่นชื่อ `thai-abc`)
2. อัปโหลดไฟล์ (หรือใช้คำสั่ง git):
   ```bash
   cd thai-abc
   git init
   git add .
   git commit -m "Thai ABC learning app"
   git branch -M main
   git remote add origin https://github.com/<ชื่อผู้ใช้>/thai-abc.git
   git push -u origin main
   ```
3. ไปที่ **Settings → Pages**
4. ที่ **Source** เลือก `Deploy from a branch`
5. เลือก branch `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**
6. รอสักครู่ เว็บจะเปิดที่ `https://<ชื่อผู้ใช้>.github.io/thai-abc/`

เท่านี้ก็พร้อมให้เด็ก ๆ เล่นได้เลย! 🎉
